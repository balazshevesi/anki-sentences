# anki-language-sentence-study-decks
A set of scripts to generate anki decks


source of sentences and translations: https://opus.nlpl.eu/

source of audio: aws polly text to speech (relatively cheap, gives timestamps, will store AAC for the best compromise between quality, file size, compatibility)

will use argos for translating single words and the most common bi-grams : https://github.com/argosopentech/argos-translate

inspiration: https://ankiweb.net/shared/info/1067616364

will use https://github.com/hermitdave/FrequencyWords/tree/master for the frequency list
